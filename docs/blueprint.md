# **App Name**: PixelPro Folio

## Core Features:

- Minimalist Navbar: A clean navigation bar with a logo, menu, and dark mode toggle.
- Typewriter Hero: Animated hero section showcasing key skills using CSS typewriter effect.
- Custom Services Carousel: A slider to display available web development packages.
- Tech Stack Showcase: Display of the technologies used (React, Node.js, etc.) using React Icons.
- Contact Form Submission: A contact form that sends messages via EmailJS.
- Interactive Map: Integration of an interactive map using Leaflet, for the contact page.
- Code quality evaluation tool: Use an AI model to analyze generated code snippets to detect instances of Firebase branding and ensure adherence to project guidelines to eliminate those brands, generating reports

## Style Guidelines:

- Primary color: White (#FFFFFF) for a clean background.
- Background color: Off-white (#F0F0F0), a slightly desaturated near-white hue that works for the light color scheme.
- Accent color: Blue (#3B82F6) for interactive elements and highlights.
- Headline font: 'Space Grotesk' (sans-serif) for headers and titles.
- Body font: 'Inter' (sans-serif) for body text, paragraphs, and descriptions.
- React Icons (Heroicons, Lucide) for general UI elements and technology icons. No Firebase branding.
- Grid-based layout for consistent spacing and alignment.
- Subtle CSS animations for transitions and interactions.
